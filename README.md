# NLP-Part-of-Speech-Tagging

A beginner Natural Language Processing (NLP) project built using Python and NLTK to perform word tokenization and Part-of-Speech (POS) tagging on text data.

## 📌 Overview

Part-of-Speech (POS) Tagging is the process of assigning grammatical categories to words in a sentence, such as nouns, verbs, adjectives, and adverbs.

This project demonstrates:

* Text preprocessing
* Word tokenization
* Part-of-Speech tagging using NLTK

## 🛠 Technologies Used

* Python
* NLTK (Natural Language Toolkit)

## 🚀 Features

* Tokenizes text into individual words.
* Assigns POS tags to each token.
* Displays grammatical categories for each word.

## 📖 Example Output

```python
[('NLP', 'NNP'),
 ('enhances', 'VBZ'),
 ('data', 'NNS'),
 ('analysis', 'NN')]
```

## 📚 POS Tag Examples

| Tag | Meaning                    |
| --- | -------------------------- |
| NN  | Noun                       |
| NNS | Plural Noun                |
| NNP | Proper Noun                |
| VBZ | Verb (3rd Person Singular) |
| JJ  | Adjective                  |
| RB  | Adverb                     |
